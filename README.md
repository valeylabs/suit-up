# Suit Up! CSS Framework
The most awesome CSS framework will have ever seen. It's legen-wait for it-dary!

## Components
* Grid
* Base Styles ("UI Kit")
  * Navigation
    * Headers
    * Menus/links
  * Forms
    * `select`, `input`, `textarea`
    * `fieldset`, `label`
    * Validation messages/styles
    * Input groups
  * Buttons
    * `button`, `a.button`
  * Tables
    * stripped, no borders
  * Badges
  * Labels
  * Alerts
  * Tabs
